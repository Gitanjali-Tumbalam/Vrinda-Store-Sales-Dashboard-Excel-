# 📊 Vrinda Store Annual Sales Dashboard (Excel)

## 📌 Project Overview

This project presents an interactive Sales Dashboard built in **Microsoft Excel** to analyze the annual sales performance of **Vrinda Store for 2022**.

The dashboard provides business insights through dynamic charts, Pivot Tables, and Slicers.

---

## 🚀 Features

- Monthly Orders vs Sales Analysis
- Sales Distribution by Gender
- Order Status Analysis
- Top 5 States by Sales
- Age Group vs Gender Analysis
- Sales by E-commerce Channels
- Interactive Filters using Slicers

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

---

## 💡 Business Insights

- Women contribute a higher percentage of sales than men.
- Most orders are successfully delivered.
- Maharashtra generated the highest sales.
- Amazon is the leading sales channel.
- Adults are the largest customer segment.
